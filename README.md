# Blu4Net
Blu4Net is a .NET library that interfaces with BluOS players. It uses an event-driven (Observable RX), non-blocking model (Async/Await) that makes it lightweight and efficient.

Supported Targets:

- .NET Standard: 2.0
- .NET Standard: 2.1
